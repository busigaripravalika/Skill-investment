# Skill Investment + Avoidance Engine

## Overview
The **Skill Investment + Avoidance Engine** is a Streamlit-based web application designed to help users make data-driven decisions about which skills to learn or avoid.

It evaluates skills using multiple factors:
- Demand
- Salary potential
- Growth trend
- Competition level
- Synergy with other skills

Based on these inputs, the system calculates a score, estimates risk, and recommends how to allocate learning time effectively.

## Features
- Multi-criteria skill evaluation
- Weighted scoring system
- Risk assessment model
- Smart time allocation
- Skill classification:
  - Invest
  - Reduce
  - Avoid
- Interactive UI using Streamlit
- Visual representation using bar charts
